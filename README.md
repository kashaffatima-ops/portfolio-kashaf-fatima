# SQA Portfolio Website

A modern and responsive Software Quality Assurance portfolio website designed to showcase professional experience, technical expertise, projects, certifications, and achievements. Built with React, TypeScript, Vite, and Tailwind CSS, the portfolio delivers a fast, engaging, and visually polished user experience with smooth animations, interactive components, and a mobile-first design approach.

This project serves as a professional online presence for QA engineers, highlighting testing skills, automation knowledge, project contributions, and career accomplishments in a clean and accessible format.
